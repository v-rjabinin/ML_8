# ML_8

Данный проект посвящён основам нейронных сетей и глубокого обучения. В рамках работы реализован многослойный персептрон (MLP) с нуля, включая алгоритм обратного распространения ошибки (backpropagation). Проект включает сравнительный анализ различных функций активации (sigmoid, ReLU, tanh, cosine), оптимизаторов (SGD, Adam) и сравнение собственной реализации с эталонными решениями из sklearn и PyTorch на данных о покупке подержанных автомобилей с Kaggle.

## ⚙️ Реализованные методы и классы

**Оптимизаторы**
- `SGD`: Стохастический градиентный спуск с возможностью экспоненциального затухания learning rate (параметры `gamma` и `step_size`);
- `Adam`: Адаптивный оптимизатор с оценкой первых и вторых моментов градиентов, bias correction и параметрами `p1`, `p2`, `eps`.

**Многослойный персептрон**
- `MLP`: Собственная реализация нейронной сети с одним скрытым слоем. Поддерживает:
  - Произвольную функцию активации для скрытого слоя (передаётся как `callable`);
  - Сигмоиду на выходном слое для бинарной классификации;
  - Методы `fit`, `predict_proba`, `predict`;
  - Инициализацию весов малыми случайными числами (нормальное распределение);
  - Mini-batch обучение с настраиваемым размером батча;
  - Валидацию данных через `sklearn.utils.check_X_y`;
  - Расчёт градиентов методом обратного распространения ошибки (backpropagation);
  - Интеграцию с различными оптимизаторами через параметр `optim_class`.

**Дополнительные классы**
- `AuctionDataset`: Класс для работы с данными в PyTorch (наследование от `torch.utils.data.Dataset`);
- `OneLayerNetwork`: Реализация MLP в PyTorch для сравнительного анализа.

<a id="-основные-задачи-проекта"></a>
## 📈 Основные задачи проекта

**Предобработка данных и Feature Engineering**
- Загрузка и очистка данных Kaggle Don'tGetKicked;
- OneHotEncoding категориальных признаков с защитой от data leakage (`handle_unknown='ignore'`, `min_frequency=24`);
- Стандартизация числовых признаков через `StandardScaler` (fit на train, transform на valid/test);
- Заполнение пропущенных значений с помощью групповой агрегации (по `Make`, `Model`, `Trim`) с использованием кастомного класса `Filler`.

**Реализация и обучение нейронной сети**
- Разработка MLP с нуля: прямой проход (forward propagation), расчёт градиентов (backpropagation), обновление весов;
- Реализация алгоритмов оптимизации: SGD с затуханием learning rate и Adam;
- Обучение с использованием бинарной кросс-энтропии (Log Loss) в качестве функции потерь;
- Достижение Gini score ≥ 0.15 на валидационной выборке (фактический результат: **0.300**).

**Сравнительный анализ моделей**
- Сравнение собственной реализации с `sklearn.MLPClassifier` (преимущество custom-реализации: **0.300** vs **0.182**);
- Исследование различных функций активации: sigmoid (**0.314**), ReLU (**0.295**), tanh (**0.300**), cosine (**0.291**);
- Реализация MLP в PyTorch и сравнение с numpy-версией (результат PyTorch: **0.308**);
- Анализ причин различий в производительности (инициализация весов, точность вычислений, диапазон гиперпараметров).

**Валидация и интерпретация результатов**
- Оценка качества на всех трёх выборках (train: **0.006**, valid: **0.308**, test: **0.282**);
- Диагностика переобучения/недообучения (модель не переобучена, но показывает признаки недообучения из-за простой архитектуры с одним скрытым слоем);
- Визуализация процесса обучения через tqdm (отслеживание loss по эпохам).
